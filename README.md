# ⚡ N'ZASSA - THE GLOBAL FEED
Créé par **NDIAYE ADAMA**

## 🚀 Installation rapide
1. `git clone https://github.com/votre-nom/nzassa-app`
2. `npm install`
3. Configure ton fichier `.env.local` avec tes clés Supabase.
4. `npm run dev` pour lancer en local.

## 🛡️ Sécurité
- Auth Wall obligatoire.
- RLS (Row Level Security) activée.
- Protection par Device ID pour les bannissements.
